# Friends Group Stories

This is a simple static website showing short friend profiles, now with a neon-styled header and subtle animated particle effects (see `script.js`).

Open the site locally by opening `index.html` in your browser, or serve it with a simple HTTP server:

PowerShell / Command Prompt:
```
python -m http.server 8000
# then open http://localhost:8000 in your browser
```
